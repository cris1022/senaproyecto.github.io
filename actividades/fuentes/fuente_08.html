Letter and word Spacing

by TPS Sena

El kerning es el termino que utiliza los tipografos para el espacio entre cada letra. Puede controlar el espacio entre cada letra con la propiedad: letter-spacing. Es particularmente util para aumentar al kerning cuando su encabezado o frase está en mayúsculas. si el texto esta en un caso normal, aumentar o disminuir el kerning puede dificultar la lectura

El intervalo intermedio

Cuando se especifica un valor para estas prpiedades, se debe dar EMS. El intervalo predeterminado entre las palbras se establece por el tipo de letra (a menudo alrededor de 0.25 EM),y es poco probable que deba cambiar esta propiedad con regularidad .
